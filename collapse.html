    The .collapse class indicates a collapsible element (a div in our example); this is the content that will be shown or hidden with a click of a button.

To control (show/hide) the collapsible content, add the data-bs-toggle="collapse" attribute to an a tag or a button element. Then add the data-bs-target="#id" attribute to connect the button with the collapsible content (
).

Accordion

Use the data-bs-parent attribute to make sure that all collapsible elements under the specified parent will be closed when one of the collapsible item is shown.
